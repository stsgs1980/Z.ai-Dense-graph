# 主题色预设（hemes） 套精心调配的主题色板,保证"电子杂志 × 电子墨水"的美学不垮。**不允许用户自定义颜色——色彩搭配错了画面瞬间变丑**,只从以下预设中挑选。---## 使用方法. 问用户选哪套(或基于内容推荐一套). 打开 `assets/template.html` 的 `style` 块. 找到开头的 `root{` 块. **整体替换**标有"主题色"注释的那几行 `--ink` / `--ink-rgb` / `--paper` / `--paper-rgb` / `--paper-tint` / `--ink-tint`. 其他  都走 `var(--...)`,无需任何其他改动---## 🖋 墨水经典 (onocle 默认)**适合**通用分享、商业发布、科技产品、任何场景都安全的默认选择。**调性**纯墨黑 + 暖米白,杂志感最强,onocle / pricot /  ook part 风。```css--ink#aab--ink-rgb,,--paper#fefea--paper-rgb,,--paper-tint#eede--ink-tint#a```---## 🌊 靛蓝瓷 (ndigo orcelain)**适合**科技/研究/数据分享、工程师文化、深度内容、技术发布会。**调性**深靛蓝 + 瓷白,冷静、理性、有深度,像学术期刊或蓝印花瓷器。```css--ink#afd--ink-rgb,,--paper#fff--paper-rgb,,--paper-tint#eeec--ink-tint#aa```---## 🌿 森林墨 (orest nk)**适合**自然/可持续/文化/非虚构内容、户外品牌、环保主题。**调性**深森林绿 + 象牙,沉稳、有呼吸感,像旧版《国家地理》。```css--ink#aef--ink-rgb,,--paper#ffe--paper-rgb,,--paper-tint#eceda--ink-tint#dc```---## 🍂 牛皮纸 (raft aper)**适合**怀旧/人文/阅读/历史/文学分享、独立杂志、手作品牌。**调性**深棕 + 暖米,像牛皮信封或老笔记本,温暖、有年代感。```css--ink#ae--ink-rgb,,--paper#eedfc--paper-rgb,,--paper-tint#edb--ink-tint#aad```---## 🌙 沙丘 (une)**适合**艺术/设计/创意/时尚分享、画廊手册、审美优先的私享会。**调性**炭灰 + 沙色,克制、高级、中性,像沙漠黄昏或建筑设计图册。```css--ink#fa--ink-rgb,,--paper#fed--paper-rgb,,--paper-tint#edbf--ink-tint#d```---## 推荐选择参考| 如果是... | 推荐主题 ||---|---|| 不知道选啥 / 第一次用 | 🖋 墨水经典 ||  / 技术 / 产品发布 | 🌊 靛蓝瓷 || 内容 / 行业观察 / 文化 | 🌿 森林墨 || 书评 / 生活方式 / 人文 | 🍂 牛皮纸 || 设计 / 艺术 / 品牌 | 🌙 沙丘 |---## 切换原则- **一份 deck 只用一套主题**,不要中途换色- eb shader 的默认主色(钛金色散 / 银色流动)适配所有  套(经测试可接受)- `currentolor` 驱动的 border / icon 会跟随 section 的 text color 自动适配,无需额外调整- 选定主题后,`title` 文字和 `chrome` 文案可以强化该主题的语义(例如牛皮纸配"ol. · 秋"这种)## ❌ 不要做的事- ❌ **不允许混搭**(例如 ink 取墨水经典的,paper 取沙丘的)——会彻底违和- ❌ **不允许用户随便给一个 hex 值**——需委婉拒绝并展示  套预设让选- ❌ **不要直接修改 template.html 其他地方的颜色**——所有散落 rgba 都走 var,改 root 一处即可选定主题后在 skill 对话中告诉用户"用 🖋 墨水经典 / 🌊 靛蓝瓷 ..."并在 deck 项目记录里备注,方便后续迭代时保持一致。
