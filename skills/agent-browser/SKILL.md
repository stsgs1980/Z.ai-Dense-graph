Navigate: `agent-browser open <url>`2. Snapshot: `agent-browser snapshot -i` (returns elements with refs like `@e1`, `@e2`)3. Interact using refs from the snapshot4. Re-snapshot after navigation or significant DOM changes## Commands### Navigation```bashagent-browser open <url>      # Navigate to URLagent-browser back            # Go backagent-browser forward         #
