# STSGS Project Rules

## Component Source
- ALWAYS import from @stsgs/ui
- NEVER create Button, Card, Dialog etc.
  from scratch — use @stsgs/ui

## Anti-Monolith
- Max 150 lines per component
- Max 3 useState per component
- No fetch in client components
- Barrel exports in every directory
- Layers: tokens → ui → sections → features

## New Components
- Check @stsgs/ui first
- If missing → add to @stsgs/ui, then import
